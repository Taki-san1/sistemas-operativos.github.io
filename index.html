Bitácora del Proyecto de Sistemas Operativos

Este proyecto tiene como objetivo aplicar de forma práctica los conocimientos adquiridos en la asignatura de Sistemas Operativos, abordando temas como la instalación de distribuciones Linux, automatización con scripts, uso de contenedores Docker y despliegues en Kubernetes.

  • Aplicar habilidades prácticas en la instalación y configuración de sistemas Linux.
  • Explorar herramientas y entornos de desarrollo modernos en Linux.
  • Diseñar scripts de automatización útiles para mantenimiento del sistema.
  • Crear y gestionar imágenes y contenedores Docker.
  • Desplegar servicios en Kubernetes usando pods y configuraciones básicas.
  • Desarrollar un sitio web que documente todas las actividades realizadas.

Portada

Este proyecto tiene como objetivo que los estudiantes de Sistemas Operativos apliquen los conocimientos adquiridos mediante la instalación de distribuciones Linux, uso de máquinas virtuales y contenedores Docker, así como la implementación de configuraciones en Kubernetes. La práctica permite una conexión entre lo teórico y el mundo profesional.

Scripts de automatización

Se desarrollaron scripts Bash para automatizar tareas comunes. Por ejemplo:

#!/bin/bash
# Script para actualizar el sistema
echo "Actualizando..."
sudo apt update && sudo apt upgrade -y
echo "Listo."

Este script permite mantener el sistema operativo actualizado de manera rápida y eficiente.
