Portada

Este proyecto tiene como objetivo que los estudiantes de Sistemas Operativos apliquen los conocimientos adquiridos mediante la instalación de distribuciones Linux, uso de máquinas virtuales y contenedores Docker, así como la implementación de configuraciones en Kubernetes. Actividades realizadas

1.1 Instalación en Ambiente Gráfico

Guía de instalación de Ubuntu en VirtualBox

A continuación, se detalla el procedimiento realizado para instalar el sistema operativo Ubuntu utilizando la plataforma de virtualización VirtualBox:

  1. Descarga de la imagen ISO de Ubuntu

    Se accedió al sitio oficial de Ubuntu a través del siguiente enlace:
    🔗 https://ubuntu.com/download/desktop

    Desde allí se seleccionó la versión más reciente del sistema operativo Ubuntu (por ejemplo, Ubuntu 24.04 LTS) y se descargó la imagen en formato .iso para su posterior instalación.

  2. Creación de la máquina virtual

    Una vez descargada la imagen, se abrió VirtualBox y se procedió a crear una nueva máquina virtual mediante los siguientes pasos:

    • Se hizo clic en el botón "Nuevo".
    • En el campo de nombre, se ingresó "ubuntu".
    • Se seleccionó la imagen ISO descargada previamente.
    • Se configuró la cantidad de memoria RAM a asignar.
    • Se definió el número de núcleos de CPU que utilizará la máquina.
    • Se completó el asistente de creación de la máquina virtual.
  3. Proceso de instalación del sistema operativo

    Con la máquina virtual ya creada, se procedió a su ejecución. Al iniciarla:

    • Se cargó el instalador gráfico de Ubuntu.
    • Se seleccionó el idioma, la zona horaria y la distribución del teclado.
    • Se configuraron los datos del usuario principal (nombre, nombre del equipo y contraseña).
    • Se siguieron las instrucciones del asistente hasta completar el proceso de instalación.

Scripts de automatización

Se desarrollaron scripts Bash para automatizar tareas comunes. Por ejemplo:

#!/bin/bash
# Script para actualizar el sistema
echo "Actualizando..."
sudo apt update && sudo apt upgrade -y
echo "Listo."

Este script permite mantener el sistema operativo actualizado de manera rápida y eficiente.
