Portada

Este proyecto tiene como objetivo que los estudiantes de Sistemas Operativos apliquen los conocimientos adquiridos mediante la instalación de distribuciones Linux, uso de máquinas virtuales y contenedores Docker, así como la implementación de configuraciones en Kubernetes. La práctica permite una conexión entre lo teórico y el mundo profesional.

Actividades realizadas

1. Instalación de Ubuntu en Máquina Virtual

Se utilizó VirtualBox para instalar Ubuntu 22.04 LTS. El proceso incluyó la creación de una máquina virtual, configuración de red y discos virtuales.

2. Comandos Básicos en Linux

Se exploraron comandos esenciales para la gestión de archivos, permisos y procesos, como ls, chmod, ps, y top.

Scripts de automatización

Se desarrollaron scripts Bash para automatizar tareas comunes. Por ejemplo:

#!/bin/bash
# Script para actualizar el sistema
echo "Actualizando..."
sudo apt update && sudo apt upgrade -y
echo "Listo."

Este script permite mantener el sistema operativo actualizado de manera rápida y eficiente.

Docker y Kubernetes

3. Creación de Imagen Docker Personalizada

Se construyó una imagen Docker con Apache y una página HTML personalizada para simular un entorno web.

4. Despliegue con Kubernetes

Se utilizó Minikube para crear un clúster local y desplegar la imagen Docker con servicios y pods configurados.
